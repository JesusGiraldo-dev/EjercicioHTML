Presentación de la actividad de crear una página con HTML.

Ejercicio 1: Cuestionario

1. ¿Qué es HTML?: Es el lenguaje de marcado estándar para crear páginas web.

2. Funciones: <html> es la raíz, <head> contiene metadatos y <body> el contenido visible.

3. Etiqueta vs Atributo: La etiqueta define el elemento y el atributo da información extra.

4. <title>: Define el nombre de la pestaña del navegador.

5. Semántica: <header>, <main> y <footer>.
